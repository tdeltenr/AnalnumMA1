Le programme possède une petite interface :  
1) Afficher ou non la solution sur gnuplot : y = yes / n = no

2) Rentrer n qui est automatiquement converti en pas valide via : m = n*8 + 1

3) Spécifier la position du radiateur : 0 pour la fenêtre /  1 pour l'autre position

4) Spécifier la valeur de rho 

5) choisir son solveur : i = itératif / d = direct / b = les deux
